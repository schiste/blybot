# Code of Conduct

This project is part of the Wikimedia movement's technical spaces. All
participation in this repository — issues, pull requests, reviews, and
discussions — is governed by the
[Wikimedia Code of Conduct for technical spaces](https://www.mediawiki.org/wiki/Code_of_Conduct).

In short: be respectful, assume good faith, and keep the space welcoming for
everyone. Report concerns as described in the linked policy, or privately to
the maintainer for repository-local matters.
